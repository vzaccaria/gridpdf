Usage:
    gridpdf NR NC [ --widthcm | -w WIDTH ] [ --griddot | -g ]
    gridpdf ( -h | --help )

Options:
    -h, --help              help for gridpdf
    -w, --widthcm WIDTH     width
    -g, --griddot           generate griddot (no text and indent lines)
Commands:

Arguments:
    NR                      number of rows
    NC                      number of columns
