Usage:
    gridpdf NR NC [ -w WIDTH ] [ --griddot | -g ] [ -d DIR ]
    gridpdf ( -h | --help )

Options:
    -h, --help              help for gridpdf
    -d, --dir DIR           destination DIR
    -w, --widthcm WIDTH     width
    -g, --griddot           generate griddot (no text and indent lines)
Commands:

Arguments:
    NR                      number of rows
    NC                      number of columns
